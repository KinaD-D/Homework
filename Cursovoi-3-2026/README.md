# Это учебный проект по созданию веб-скрейпера сайта компании "Электросила"

Также, помимо веб-скрейпера здесь в качестве учебного прогресса будут продемонстрированы:

* Ранее изученные в ходе лабораторных библиотеки. Для демонстрации прогресса учёбы и закрепления материала.

* Хорошая структура кода с упором на "питоновскую" читабельность с комментариями. Для закрепления правил PEP8

* Разделённая на отдельные файлы и каталоги программа. Что способствует дальнейшему "теоретическому"
расширению этой программы, модульности и следующему за этим снижению связности компонентов, что приводит к более удобной 
отладке, тестированию и написанию кода. PS: тесты по структуре повторяют код, чтобы было понятно,
на какую часть кода сделаны тесты, а на какую нет

* И много других приятных мелочей, вроде использования тестов и создания файлов вроде "LICENSE"

#### В общем всё, что демонстрирует крутого программиста

## Установка
1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/Kinad-d/Homework/Cursovoi-3-2026.git
    cd cursovoi-3-2026
    ```
    
2.  Установите необходимые библиотеки:
    ```bash
    pip install -r requirements.txt
    ```
    
## Структура проекта
- `src`: Основной код.
  - `database`: Здесь находятся необходимые модули для работы с БД.
  - `main.py`: Модуль запуска всего кода.
- `tests`: Тесты smoke & acceptance.
- `requirements.txt`: Список зависимостей.
